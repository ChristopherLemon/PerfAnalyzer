General Info

Browser based user interface, for performance profiling with the Linux perf hardware counters.

Browser Support

By default, the application launches with the system default browser. This can be changed by providing the path to an alternative browser as a command line argument (i.e. --browser=path_to_browser). The recommended browser is Chrome, or Firefox.

Operating System

Due to the dependence on perf, the performance profiling can only be performed on a Linux system. However, the user interface can be run on Windows or Linux. When running the user interface on Windows the Secure Shell (SSH) option must be used, to perform the performance profiling on a remote Linux System. {% endblock %}
